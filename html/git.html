Git

  • git init - Создание репозитория
  • git clone - Локализация\Клонирование репозитория из GitHub на компьютер через HTTPS ссылку.
  • git status - Проверка наличия файлов и их статуса взаимодействия с удалённым репозиторием в GitHub.
  • git add - Добавление файла в статус отслеживаемых GitHub.
  • git commit -m - Сохранение изменений в Git.
  • git push - Отправка изменений на Удалённый репозиторий в GitHub.
  • git pull - Получение изменений из удалённого репозитория с GitHub на локальный.
  • git log - Просмотр истории коммитов в репозитории.
  • git branch - Создание копии Мастер-ветки для альтернативной разработки проекта.
  • git git checkout - Переключение на ветку
